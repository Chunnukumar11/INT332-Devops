# Basic Docker Commands

## Check Docker Version
docker version

## Download Image
docker pull ubuntu

## Show All Images
docker images

## Run Container
docker run ubuntu

## Run Container in Interactive Mode
docker run -it ubuntu bash

## Show Running Containers
docker ps

## Show All Containers
docker ps -a

## Stop Container
docker stop container_id

## Remove Container
docker rm container_id

## Remove Image
docker rmi image_id
