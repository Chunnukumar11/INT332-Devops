# Unit 1 – Docker Basics

## DevOps
DevOps combines Development and Operations to improve collaboration and automate software delivery.

## Containers
Containers are lightweight environments that package applications with all dependencies.

Advantages:
- Fast startup
- Portable
- Efficient resource usage

## Docker
Docker is a platform used to build and run containers.

## Docker Image
A Docker image is a template used to create containers.

Example:
docker pull ubuntu

## Docker Container
A container is a running instance of an image.

Example:
docker run ubuntu
